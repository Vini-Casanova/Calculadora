## Calculadora Desenvolvida em java

# Tecnologias usadas:
- Java 18.0.2
-Bibliotecas:
-Java Swing
-Java Awt

## Como Usar?
-Tenha Instalado Jdk
-Baixe o arquivo e inicio App.js na sua IDE
